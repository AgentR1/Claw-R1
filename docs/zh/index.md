# Claw-R1 中文文档

Claw-R1 是面向 Agentic Reinforcement Learning 的数据基础设施，关注从 Agent 交互中采集、评估、筛选和供给训练数据。

当前默认文档语言为英文。中文入口保留项目概览，并可通过页面顶部语言切换返回英文文档。

## 最新特性

- Dashboard 已接入真实 Ray DataPool 和参数同步 actor，用于 Agentic RL data lifestyle management。
- Dashboard 覆盖采集、表示、筛选、prefix-tree 优化预览和训练消费状态。
- Prefix Tree Merge 用于减少 multi-step agent 训练中的共享前缀重复计算。
